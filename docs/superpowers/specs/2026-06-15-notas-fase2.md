## 2. Límite de 20 MB del Bot API de Telegram
- El Bot API descarga archivos del lado del bot hasta **20 MB**. Los reels cortos suelen entrar.
- Para HD/largos que se pasen: **Local Bot API server** (sube el límite a ~2 GB) o **cliente
  MTProto** (Telethon/Pyrogram, cuenta usuario en vez de bot).
- ⚠️ Webhooks NO cambia el límite de tamaño (es solo cómo se reciben los updates) — no sirve para esto.

## 3. Verificar ffmpeg
- Hermes instala ffmpeg, pero antes de procesar video en Fase 2, verificar en la terminal de Sergio:
  `ffmpeg -version`. Si no lo reconoce, agregarlo al PATH.

## 4. Ingesta por link (alternativa al archivo)
- Además de mandar el archivo, se puede mandar un **link** (IG/YT/TikTok) y bajarlo del lado del
  servidor con **yt-dlp**. Cómodo, pero el scraping de IG puede romperse. Conviven ambos caminos.
