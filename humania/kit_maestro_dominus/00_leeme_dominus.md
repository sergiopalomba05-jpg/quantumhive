# Kit Maestro de Dominus

Este kit existe para mantener a Dominus consistente como avatar oficial de QuantumHive.

Dominus no se debe generar desde cero para cada pieza. Dominus es el rostro persistente de la marca: el presentador, guia y representante visual de la infraestructura QuantumHive.

## Objetivo

- Mantener una identidad visual unica.
- Ordenar las imagenes oficiales y descartadas.
- Separar el avatar del logo real.
- Crear reglas claras para video, subtitulos, poses y nomenclatura.
- Evitar que cada nuevo contenido rompa la identidad de Dominus.

## Enfoque actual

Primero 2.5D.

El 3D completo queda para una etapa futura. Para videos comerciales, landings y reels, lo mas importante ahora es que Dominus funcione como presentador persistente en primer plano, medio cuerpo y cuerpo completo.

## Importante sobre fondos

No crear todavia fondos definitivos dentro del kit principal.

Los fondos se van a generar despues por producto o por video:

- Mesero Virtual.
- Barberia.
- Tienda.
- QuantumHive institucional.
- WhatsApp delivery.
- Refacciones o servicios.

El kit principal se enfoca en Dominus como avatar oficial, no en almacenar todos los universos visuales de producto.

## Formula de contenido

```text
Dominus persistente + clips de apoyo + subtitulos grandes + demo visual
```

Lo que cambia:

- producto
- rubro
- fondo
- demo
- copy
- CTA

Lo que no cambia:

- rostro oficial
- silueta general
- estilo premium
- paleta base
- rol de presentador
