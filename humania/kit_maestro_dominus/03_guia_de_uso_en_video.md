# Guia de uso en video

Dominus debe funcionar como presentador persistente.

## Estructura base de cada video

```text
1. Gancho
2. Problema real
3. Solucion con empleado virtual
4. Demo o ejemplo visual
5. Beneficio comercial
6. Cierre con QuantumHive
7. CTA
```

## Donde aparece Dominus

Dominus puede aparecer:

- al inicio para dar autoridad
- en el medio para explicar
- al cierre para CTA
- sobre fondos distintos segun producto

No usar siempre el mismo fondo.

Mantener el avatar consistente y variar el contexto.

## Uso por encuadre

Primer plano:

- gancho fuerte
- frase institucional
- portada de reel
- momento de autoridad

Medio cuerpo:

- explicacion principal
- comercial corto
- presentacion de beneficios
- senalar texto, graficos o demo

Cuerpo completo:

- landing principal
- portada de producto
- banner
- apertura o cierre institucional

## Ejemplos de CTA

```text
Proba una demo segun tu rubro
Conoce el Mesero Virtual
Crea un empleado virtual para tu negocio
Habla con QuantumHive
```
