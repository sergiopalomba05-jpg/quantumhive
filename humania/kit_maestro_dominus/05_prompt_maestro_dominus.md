# Prompt maestro de Dominus

Usar este prompt como base cuando haga falta generar nuevas poses de Dominus.

## Prompt base

```text
Crear una imagen premium ultra realista de Dominus, avatar oficial de QuantumHive. Hombre cibernetico elegante, pelo largo peinado hacia atras y hacia un costado, rostro serio y confiable, traje negro biomecanico de alta tecnologia con detalles dorados, estilo humano dominante con tecnologia integrada de forma sutil en rostro, cuello, brazos y traje. Estetica negra y dorada, presencia de autoridad, lujo, inteligencia artificial empresarial, infraestructura multiagente, sin abeja, sin insectos, sin casco, sin caricatura, sin texto pequeno en la ropa, sin logo deformado. Fondo transparente real o fondo blanco limpio segun se indique. Composicion cinematografica, iluminacion premium, textura detallada, mirada directa, estilo serio y profesional para videos comerciales e institucionales de QuantumHive.
```

## Agregados por pose

Cuando se necesite una pose, agregar al final:

```text
pose: medio cuerpo senalando hacia la izquierda
pose: medio cuerpo senalando hacia la derecha
pose: medio cuerpo senalando hacia abajo para CTA
pose: medio cuerpo explicando con ambas manos abiertas
pose: primer plano serio mirando a camara
pose: cuerpo completo institucional
```

## Prompt negativo

```text
cara distinta, pelo distinto, estilo caricatura, abeja, insecto, casco, villano exagerado, superheroe infantil, robot frio, personaje de videojuego barato, influencer generico, texto pequeno en la ropa, logo deformado, manos deformes, dedos extra, baja calidad, imagen borrosa, identidad inconsistente
```

## Regla de consistencia

Antes de aceptar una nueva imagen, comparar:

- rostro
- cabello
- edad visual
- silueta
- traje
- nivel de robotizacion
- presencia premium

Si falla en rostro o manos, no entra al asset principal.
