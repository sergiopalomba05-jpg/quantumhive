# Nombres de archivos

Usar nombres en espanol, sin tildes ni caracteres especiales.

Usar minusculas, guiones bajos y version al final.

## Formato general

```text
dominus_[encuadre]_[pose_o_expresion]_v01.png
```

## Primer plano

```text
dominus_primer_plano_serio_v01.png
dominus_primer_plano_sonrisa_confiada_v01.png
```

Uso:

- videos institucionales
- presentaciones fuertes
- portadas
- reels donde Dominus abre el video
- momentos de autoridad

## Cuerpo completo

```text

```

Uso:

- landing principal
- portada de producto
- piezas institucionales
- presentaciones
- banners
- inicio o cierre de videos

## Versionado

- `v01`: primera version oficial usable.
- `v02`: correccion o mejora menor.
- `v03`: nueva version aprobada.

No reemplazar archivos oficiales sin guardar la version anterior.
