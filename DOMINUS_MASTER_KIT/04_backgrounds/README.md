# Fondos maestros

Dominus puede aparecer en distintos contextos, pero todos deben conservar la estetica QuantumHive.

## Fondos base

### QuantumHive Control Room

Uso: institucional, presentador principal, videos de marca.

Visual:

- Centro de control oscuro.
- Panales dorados sutiles.
- Pantallas de datos.
- Estetica premium.
- Nada caricaturesco.

### Tech Neutral Background

Uso: reels generales, explicaciones rapidas, piezas limpias.

Visual:

- Fondo oscuro limpio.
- Detalles tecnologicos sutiles.
- Espacio para subtitulos grandes.

### Restaurant / Mesero Virtual Background

Uso: Mesero Virtual, gastronomia, QR inteligente.

Visual:

- Restaurante premium.
- Mesas, QR, hologramas o interfaz.
- Sensacion comercial real.

### Landing Hero Background

Uso: landing principal y secciones hero.

Visual:

- Cinematografico.
- Premium.
- Con espacio para copy y CTA.
