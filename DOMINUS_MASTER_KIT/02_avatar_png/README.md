# Assets de avatar PNG

Guardar aca las versiones finales de Dominus con fondo transparente.

## Categorias

- `closeup`: primer plano.
- `bust`: pecho para arriba.
- `halfbody`: medio cuerpo.
- `three_quarter`: tres cuartos.
- `fullbody`: cuerpo completo.

## Poses minimas obligatorias

- neutral_front
- arms_crossed
- open_hand_presenting
- pointing_left
- pointing_right
- pointing_down_cta
- explaining_two_hands
- confident_relaxed
- hero_stance
- closeup_speaking

## Formato de nombre

```text
dominus_[encuadre]_[pose]_[expression]_v01.png
```

Ejemplos:

```text
dominus_closeup_serious_v01.png
dominus_halfbody_armscrossed_confident_v01.png
dominus_halfbody_pointingleft_explaining_v01.png
dominus_halfbody_pointingdown_cta_v01.png
dominus_fullbody_heroic_confident_v01.png
```
