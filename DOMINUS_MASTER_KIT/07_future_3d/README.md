# Futuro 3D

El 3D completo no es la prioridad inmediata.

Primero se consolida Dominus 2.5D como presentador persistente.

## Cuando pasar a 3D

Pasar a 3D cuando ya existan:

- Rostro oficial cerrado.
- Poses 2.5D aprobadas.
- Fondos maestros aprobados.
- Logo separado.
- Plantilla de video funcional.
- Primeros videos comerciales validados.

## Objetivo futuro

Crear una version 3D de Dominus para:

- Universo QuantumHive.
- Escenas institucionales.
- Interacciones con productos.
- Experiencias web.
- Animaciones mas complejas.
