# Logos QuantumHive

El logo debe mantenerse separado del avatar.

## Assets necesarios

- quantumhive_logo_gold.png
- quantumhive_logo_white.png
- quantumhive_isotype_gold.png
- quantumhive_logo.svg
- quantumhive_logo_horizontal.png
- quantumhive_logo_transparent.png

## Reglas

- No depender de logos generados dentro del traje.
- No usar texto pequeno sobre el pecho.
- Usar overlay limpio en edicion.
- Usar logo en cierre, esquina o placa segun pieza.
- Mantener versiones dorada, blanca y transparente.
