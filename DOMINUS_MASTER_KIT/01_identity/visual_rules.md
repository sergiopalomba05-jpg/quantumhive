# Reglas visuales

## Reglas obligatorias

- Mantener el mismo rostro oficial.
- Mantener el mismo estilo de cabello.
- Mantener la misma estetica de traje negro tecnologico.
- Mantener detalles dorados sutiles.
- Usar fondos premium oscuros o comerciales, segun contexto.
- Separar siempre al avatar del logo real.

## Evitar

- No generar veinte versiones parecidas de Dominus.
- No usar logos con letras pequenas incrustadas en el traje.
- No usar fondos caricaturescos.
- No usar abejas literales como elemento central.
- No exagerar la robotizacion.
- No cambiar color de pelo, edad, rostro o silueta.
- No usar poses comicas para piezas institucionales.

## Encuadres principales

1. Primer plano: autoridad, hooks, frases fuertes.
2. Busto: explicaciones, piezas institucionales.
3. Medio cuerpo: presentador principal para reels y demos.
4. Tres cuartos: piezas premium y portadas.
5. Cuerpo completo: landing, piezas hero y futuros assets 3D.

## Fondos permitidos

- Centro de control QuantumHive.
- Fondo tecnologico neutro.
- Restaurante premium para Mesero Virtual.
- Hero landing cinematografico.
- Fondos por rubro solo si mantienen el lenguaje visual de marca.
