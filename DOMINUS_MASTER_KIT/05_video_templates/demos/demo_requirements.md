# Requisitos para demos visuales

## Lo que debe mostrar una demo

- Producto real o mockup realista.
- Accion clara del usuario.
- Resultado visible.
- Beneficio comercial entendible.

## Demo Mesero Virtual

Mostrar:

- Cliente escanea QR.
- Menu inteligente abre.
- Sistema recomienda.
- Pedido o intencion queda guiada.
- Negocio recibe datos o metricas.

## Regla

La demo no debe ser solo estetica. Tiene que demostrar una accion concreta.
