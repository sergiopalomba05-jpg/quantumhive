# DOMINUS MASTER KIT

Kit maestro para convertir a Dominus en el avatar persistente de QuantumHive.

Este kit no es una carpeta de imagenes sueltas. Es el sistema base para que Dominus mantenga la misma identidad en videos, landings, reels, demos, portadas y piezas comerciales.

## Objetivo

- Congelar una identidad visual unica de Dominus.
- Evitar generar un personaje distinto en cada pieza.
- Separar avatar, fondos, logo, subtitulos, voz y plantillas.
- Crear una base reusable para Mesero Virtual, QuantumHive institucional y futuras ramas.

## Principio central

Dominus no se genera desde cero cada vez.

Dominus es el presentador oficial. Lo que cambia es el tema, el fondo, el copy, la demo y el contexto comercial.

Formula de contenido:

```text
Dominus persistente + clips de apoyo + subtitulos grandes + demo visual
```
