# Repositorios de Referencia para ComfyUI

## Repos Guardados como Guía de Agentes

### 1. Cubiq ComfyUI Workflows
- **URL:** https://github.com/cubiq/ComfyUI_Workflows
- **Uso:** Colección de workflows pre-armados para ComfyUI
- **Contenido:** Workflows para generación de imagen, video, control, upscale, etc.
- **Cuándo usarlo:** Cuando necesités armar un workflow nuevo o buscar inspiration

### 2. Comfy-Org Workflow Templates
- **URL:** https://github.com/Comfy-Org/workflow_templates
- **Uso:** Templates oficiales de ComfyUI para diferentes tareas
- **Contenido:** Templates para SDXL, Flux, video, audio, etc.
- **Cuándo usarlo:** Cuando necesités un template oficial y probado

## Cómo usar estos repos con agentes

Los agentes pueden:
1. Clonar estos repos localmente para tener acceso offline
2. Leer los archivos JSON de workflows para entender la estructura
3. Reutilizar patrones de nodos para crear workflows nuevos
4. Referenciar los README para entender qué hace cada nodo

## Descarga local (opcional)

```powershell
# En la VM GPU
cd C:\Users\sergio\ComfyUI-Workflows
git clone https://github.com/cubiq/ComfyUI_Workflows.git
git clone https://github.com/Comfy-Org/workflow_templates.git
```
