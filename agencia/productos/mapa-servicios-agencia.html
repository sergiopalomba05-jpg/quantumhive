INVESTIGACIÓN DE MARCA

Motor de onboarding

El Agente Investigador toma un handle de Instagram y produce un perfil.md completo: datos del negocio, tono de voz, vocabulario, valores, estética, FAQ de la audiencia. Es producto interno (alimenta cada bot) y también vendible como auditoría de marca standalone.

Qué incluye

  • Scraping respetuoso del perfil públicoBio, posts, captions, comentarios, highlights — con rate limits, nunca agresivo
  • Transcripción de los reels con más alcance
  • Análisis de identidad: tono, frases propias, diferencial, público
  • Documento perfil.md entregable + notas para system prompt

Herramientas

Apify · IG scraper Gemini 1.5 Pro · análisis masivo yt-dlp · descarga de reels Whisper · transcripción Python · orquestación

Reglas del servicio

  • Nunca usar posteos como fuente de precios/horarios
  • Output = insumo, no verdad absoluta
  • Cuenta privada → se pide acceso o data manual
Modelo de cobroIncluido en el setup de cualquier bot/servicio. Standalone: auditoría de marca + informe de esencia como producto de entrada de bajo ticket.

ASISTENTE IN-APP (TIPO TIMÓN)

Componente reusable

El asistente que vive adentro de la app o web del cliente: responde por texto, audio y foto, recuerda a cada usuario, y guía la experiencia. La versión productizada de Timón, el agente de Directimport.

Qué incluye

  • Chat multimodal: texto, notas de voz transcriptas, fotosEj.: el usuario manda una foto y el asistente encuentra el producto en el catálogo
  • Respuestas opcionales en audio
  • Memoria por usuarioCada usuario tiene su historial y preferencias, aisladas por ID
  • Captura de demanda: lo que el usuario pidió y no había, queda registrado
  • Animación de orbe estilo Jarvis para el estado de audio

Herramientas

Claude API · razonamiento y visión Whisper / Groq · transcripción de voz Supabase · memoria por user ID R3F · orbe animado

Especificaciones clave

  • Se vende como módulo dentro de apps/webs, no suelto
  • Mismo motor para cualquier vertical: catálogo, turnos, soporte
Modelo de cobroAdd-on sobre el proyecto de app/web: suma al setup + sube el mensual (consume API por uso).

Genera los documentos legales de cara al usuario para cualquier app o web: términos y condiciones, política de privacidad, acuerdo de suscripción, política de devoluciones — fundados en ley argentina (defensa del consumidor, protección de datos), con la jurisdicción como parámetro.

Qué incluye

  • Set completo de documentos por productoT&C, privacidad, suscripción, devoluciones
  • Clickwrap antes de pagarEl usuario acepta antes de cualquier cobro — protege al cliente
  • Asistente nativo que responde dudas legales dentro de la appCon los T&C como conocimiento; deriva lo que no está escrito

Herramientas

Claude API · generación y consulta Supabase · versiones y aceptaciones

Límites declarados (esto vende confianza)

  • Cubre lo de cara al usuario, no contratos societarios

Modelo de cobroIncluido en proyectos de app/web (diferencial frente a competencia que entrega apps sin legales). Standalone: paquete de documentos para apps existentes.
